# **Introduction: What IS XML?**

It focuses on carrying data,  
concentrating on what the data is, sharing it efficiently and in a way that is easy for people and machines to understand.

---

# DifferenceWithHTML

It differs from HTML, which focuses on displaying data,  
concentrating more on appearance.

- XML is a markup language used to **store and transport data**.  
- HTML is more concerned with **displaying data**.
