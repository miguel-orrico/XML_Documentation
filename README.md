# XML (eXtensible Markup Language): 
Fue diseñado para almacenar y transportar datos.

XML NO HACE NADA, surgió de la necesidad de intercambiar datos y que fuese intelligible. Por ello este lenguaje se encarga de contener información envuelto en etiquetas, las cuales no están predefinidas como HTML, sino que son inventadas por el autor del documento.
